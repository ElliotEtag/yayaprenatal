# Yaya Prenatal — Website

Die Website für Yaya Prenatal: https://yayaprenatal.de

## Für Nina: So funktioniert's

Du benutzt **Claude Code** in VS Code, um die Website zu bearbeiten. Du musst keinen Code schreiben — sag einfach, was du ändern willst, und Claude macht das für dich.

### Beispiele, was du sagen kannst

- "Ändere die Telefonnummer im Footer"
- "Aktualisiere die Preise bei den Paketen"
- "Füge einen neuen Erfahrungsbericht hinzu"
- "Ändere den Text im Impressum"
- "Zeig mir wie die Seite gerade aussieht"
- "Mach die Änderungen live"

### Wie Änderungen auf die echte Website kommen

1. Du sagst Claude, was sich ändern soll
2. Claude macht die Änderung
3. Du kannst eine **Vorschau** sehen, bevor es live geht (http://localhost:4321)
4. Wenn du zufrieden bist, sag Claude: "Mach es live"
5. Nach 1-2 Minuten ist die Änderung auf https://yayaprenatal.de sichtbar

### Wichtig zu wissen

- Nichts geht live, ohne dass du es bestätigst
- Du kannst jede Änderung rückgängig machen
- Im Zweifel: Erst Vorschau anschauen, dann live schalten
